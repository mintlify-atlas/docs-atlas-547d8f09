<div className="mt-16 mb-16 max-w-5xl mx-auto px-6">
  <h2 className="text-3xl font-bold text-gray-900 dark:text-white mb-4">Quick start</h2>
  <p className="text-base text-gray-600 dark:text-gray-400 mb-8">
    Get your XLRS system up and running in minutes
  </p>
  
  <Steps>
    <Step title="Gather hardware">
      You'll need three Raspberry Pi Pico boards (one Pico W for RX), three SX1280 radio modules, and supporting components like joysticks and an OLED display for the RC board.
      
      See the [hardware requirements](/hardware-requirements) for the complete list.
    </Step>
    
    <Step title="Install PlatformIO">
      XLRS uses PlatformIO for building and flashing firmware.
      
      ```bash
      # Install PlatformIO Core
      pip install platformio
      ```
      
      Clone the repository:
      
      ```bash
      git clone https://github.com/xlrs-dev/xlrs.git
      cd xlrs
      ```
    </Step>
    
    <Step title="Build and flash firmware">
      Build and upload firmware to each module:
      
      ```bash
      # Flash the RX module (receiver on drone)
      pio run -e rx_sx128x -t upload
      
      # Flash the TX module (transmitter)
      pio run -e tx_sx128x -t upload
      
      # Flash the RC board (controller)
      pio run -e rc-crsf -t upload
      ```
    </Step>
    
    <Step title="Pair TX and RX">
      Hold the BOOTSEL button on the RX module for 5 seconds to enter pairing mode. Power on the TX module — it will automatically pair with the RX using the default binding phrase.
      
      Once paired, the system establishes a secure encrypted connection. See [pairing configuration](/configuration/pairing) for details.
    </Step>
  </Steps>
</div>

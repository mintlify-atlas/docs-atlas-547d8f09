## Overview

The `SX128xLink` class provides a simplified interface for using the SX128x radio module (2.4GHz LoRa/FLRC) in both transmitter and receiver applications. It wraps the RadioLib library and handles hardware initialization, packet transmission, and reception.

## Class Definition

Location: `lib/SX128xLink/SX128xLink.h`

```cpp
class SX128xLink {
public:
    SX128xLink();
    bool begin();
    bool send(const uint8_t* data, size_t len);
    bool receive(SX128xPacket& packet);
    void startReceive();
    bool isReady() const { return ready; }
};
```

## Data Structures

### SX128xPacket

Container for received radio packets with signal quality metrics.

```cpp
struct SX128xPacket {
    uint8_t data[64];   // Packet payload
    size_t length;      // Actual payload length
    int16_t rssi;       // Received Signal Strength Indicator (dBm)
    float snr;          // Signal-to-Noise Ratio (dB)
};
```

### receive()

Receives a packet from the radio if available.

```cpp
bool receive(SX128xPacket& packet);
```

<ParamField path="packet" type="SX128xPacket&">
  Reference to packet structure to fill with received data
</ParamField>

<ParamField path="return" type="bool">
  Returns `true` if packet received, `false` if no packet available
</ParamField>

**Example:**

```cpp
SX128xPacket pkt;
if (radio.receive(pkt) && pkt.length >= 1) {
    Serial.print("Received packet, RSSI: ");
    Serial.print(pkt.rssi);
    Serial.print(" dBm, SNR: ");
    Serial.println(pkt.snr);
    
    uint8_t type = pkt.data[0];
    const uint8_t* payload = pkt.data + 1;
    size_t payload_len = pkt.length - 1;
    
    // Process packet based on type
    if (type == MSG_CHANNELS) {
        Protocol::decodeFrame(payload, channels, &sequence, 
                              &security, &lastSequence, pairedTxDeviceId);
    }
}
```

## Configuration

The radio module can be configured via compile-time defines:

```cpp
#define SX128X_FREQ_MHZ 2420.0f           // Frequency in MHz
#define SX128X_BW_KHZ 812.5f              // Bandwidth in kHz
#define SX128X_SF 7                       // Spreading Factor
#define SX128X_CR 5                       // Coding Rate (4/5)
#define SX128X_OUTPUT_POWER_DBM 10        // TX power in dBm
#define SX128X_FLRC_BR_KBPS 1300          // FLRC bit rate (1.3 Mbps)
#define SX128X_FIXED_PACKET_LEN 33        // Fixed packet length (0 = variable)
```

### Pin Configuration

## Overview

The Protocol library provides static functions for encoding and decoding secure radio packets, along with pairing and synchronization functionality. It integrates with the Security class for encryption and authentication.

## Protocol Class

Location: `lib/Protocol/Protocol.h`

```cpp
class Protocol {
public:
    // Frame encoding/decoding
    static void encodeFrame(uint16_t channels[NUM_CHANNELS], uint16_t sequence,
                           Security* security, const uint8_t* deviceId, 
                           uint8_t* frame);
    
    static bool decodeFrame(const uint8_t* frame, uint16_t channels[NUM_CHANNELS],
                           uint16_t* sequence, Security* security, 
                           uint16_t* lastSequence, const uint8_t* expectedDeviceId);
    
    static bool validateFrame(const uint8_t* frame);
    
    // Discovery protocol
    static void createDiscoveryPacket(const IPAddress& ip, char* buffer, 
                                     size_t bufferSize);
    static bool parseDiscoveryPacket(const char* packet, IPAddress& ip);
    
    // Sync packets
    static void createSyncPacket(uint16_t syncSeq, uint32_t timestamp,
                                Security* security, const uint8_t* deviceId,
                                uint8_t* packet, size_t* packetLen);
    
    static bool parseSyncPacket(const uint8_t* packet, size_t packetLen,
                               uint16_t* syncSeq, uint32_t* timestamp,
                               Security* security, uint8_t* sourceDeviceId);
    
    static void createSyncAck(uint16_t syncSeq, uint32_t timestamp,
                             Security* security, const uint8_t* deviceId,
                             uint8_t* packet, size_t* packetLen);
    
    static bool parseSyncAck(const uint8_t* packet, size_t packetLen,
                            uint16_t* syncSeq, uint32_t* timestamp,
                            Security* security, uint8_t* sourceDeviceId);
};
```

## Protocol Methods

### encodeFrame()

Encodes channel values into a secure binary frame with encryption and HMAC.

```cpp
static void encodeFrame(uint16_t channels[NUM_CHANNELS], uint16_t sequence,
                       Security* security, const uint8_t* deviceId, 
                       uint8_t* frame);
```

<ParamField path="channels" type="uint16_t[8]">
  Array of 8 channel values (1000-2000 µs)
</ParamField>

<ParamField path="sequence" type="uint16_t">
  Packet sequence number for replay protection
</ParamField>

<ParamField path="security" type="Security*">
  Pointer to Security instance for encryption/HMAC
</ParamField>

<ParamField path="deviceId" type="const uint8_t*">
  8-byte device ID of sender
</ParamField>

<ParamField path="frame" type="uint8_t*">
  Output buffer (must be at least FRAME_SIZE = 30 bytes)
</ParamField>

**Example:**

```cpp

uint8_t frame[FRAME_SIZE];

```

## Frame Format

Channel data frame (30 bytes total):

```
[Device ID (8)][Channel Data Encrypted (16)][Sequence (2)][HMAC (4)]
```

- **Device ID:** 8-byte sender device identifier
- **Channel Data:** 16 bytes (8 channels × 2 bytes), AES-128 encrypted
- **Sequence:** 2-byte packet sequence number
- **HMAC:** 4-byte truncated HMAC-SHA256

## Pairing Process

### Transmitter Side

1. Enter pairing mode (automatically if not paired)
2. Send `MSG_PAIRING` packets with binding UID and pairing key
3. Wait for `MSG_PAIRING_ACK` from receiver
4. Store RX device ID and exit pairing mode

### Receiver Side

1. Hold BOOTSEL button for 5 seconds to enter pairing mode
2. Listen for `MSG_PAIRING` packets
3. Verify binding UID matches
4. Store TX device ID and pairing key
5. Send `MSG_PAIRING_ACK`
6. Exit pairing mode

## Complete Usage Example

### Transmitter

From `src/tx_main_sx128x.cpp`:

```cpp
Security security;
SX128xLink radio;
uint8_t txDeviceId[DEVICE_ID_SIZE];
uint8_t pairedRxDeviceId[DEVICE_ID_SIZE];
bool hasPairedRxId = false;
bool pairingMode = false;
uint16_t sequenceNumber = 0;

void setup() {
    security.begin();
    security.getDeviceId(txDeviceId);
    
    if (security.getPairedDeviceId(pairedRxDeviceId)) {
        hasPairedRxId = true;
        Serial.println("Already paired");
    } else {
        pairingMode = true;
        Serial.println("Entering pairing mode");
    }
    
    radio.begin();
}

void loop() {
    // Handle pairing
    if (pairingMode) {
        Pairing::sendPairingKey(&radio, &security, txDeviceId,
                               &pairingMode, &lastPairingSend);
    }
    
    // Send channel data when paired
    if (hasPairedRxId && !pairingMode) {
        uint8_t frame[FRAME_SIZE];
        Protocol::encodeFrame(channels, sequenceNumber++, 
                             &security, txDeviceId, frame);
        
        uint8_t packet[FRAME_SIZE + 1];
        packet[0] = MSG_CHANNELS;
        memcpy(packet + 1, frame, FRAME_SIZE);
        radio.send(packet, FRAME_SIZE + 1);
    }
}
```

### Receiver

From `src/rx_main_sx128x.cpp`:

```cpp
Security security;
SX128xLink radio;
uint8_t rxDeviceId[DEVICE_ID_SIZE];
uint8_t pairedTxDeviceId[DEVICE_ID_SIZE];
bool hasPairedTxId = false;
bool pairingMode = false;
uint16_t lastSequence = 0;
uint16_t channels[8];

void setup() {
    security.begin();
    security.getDeviceId(rxDeviceId);
    
    if (security.getPairedDeviceId(pairedTxDeviceId)) {
        hasPairedTxId = true;
        Serial.println("Already paired");
    }
    
    radio.begin();
}

void loop() {
    // Check for pairing button
    Pairing::checkBootselButton(&pairingMode, &pairingModeStartTime, 
                               hasPairedTxId);
    
    // Receive packets
    SX128xPacket pkt;
    if (radio.receive(pkt) && pkt.length >= 1) {
        uint8_t type = pkt.data[0];
        const uint8_t* payload = pkt.data + 1;
        size_t payload_len = pkt.length - 1;
        
        switch (type) {
            case MSG_CHANNELS:
                if (hasPairedTxId && payload_len >= FRAME_SIZE) {
                    uint16_t sequence;
                    if (Protocol::decodeFrame(payload, channels, &sequence,
                                             &security, &lastSequence,
                                             pairedTxDeviceId)) {
                        // Channels received successfully
                    }
                }
                break;
                
            case MSG_PAIRING:
                Pairing::handlePairingPacket(payload, payload_len, &security,
                                            &radio, rxDeviceId, pairedTxDeviceId,
                                            &hasPairedTxId, &pairingMode);
                break;
        }
    }
}
```
